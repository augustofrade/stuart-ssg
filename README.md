# Stuart SSG

Stuart is a Static Site Generator that lets you write markdown documents and convert them into fully-fledged static websites.

This repository is a monorepo containing all Stuart projects and tools:

## @stuart-ssg/lib

Package containing both core functionality and end-user interfaces.

## @stuart-ssg/cli

Easy-to-use CLI for Stuart operations, such as project creation, page creation and building etc.

## @stuart-ssg/content

Default themes, schematics and blueprints for Stuart. Included in the `@stuart-ssg/cli` installation.
